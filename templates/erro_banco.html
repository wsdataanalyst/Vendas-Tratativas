Checklist no Render → Environment

  1. Apague DATABASE_URL (se existir)
  2. SUPABASE_MODE = direct
  3. SUPABASE_PROJECT_REF = Reference ID (Supabase → Settings → General)
  4. SUPABASE_DB_PASSWORD = senha do banco (Database → Reset password se precisar)
  5. Não precisa de SUPABASE_DB_HOST no modo direct

Save → Manual Deploy → aguarde 2 min → abra o site de novo
